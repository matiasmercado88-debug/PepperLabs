// 02 — El estudio

Treinta años haciendo software. Ahora con diseño y contenido.

PepperLabs es la evolución de Software Express, el estudio que Carlos Buira fundó en Buenos Aires en 1992. Después de tres décadas haciendo sistemas para PyMEs argentinas, sumamos diseño e identidad para cerrar el círculo: el mismo equipo se ocupa de cómo tu negocio funciona, cómo se ve y cómo se comunica.

El sistema se adapta al negocio. Nunca al revés.

— Carlos Buira · Software Express, 1992
